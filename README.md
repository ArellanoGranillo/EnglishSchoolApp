EnglishSchoolApp
This is a simple Java exercise designed to practice if-else statements.

📚 Description
The program asks the user to enter:

Student name
Student age
Based on the age provided, the program confirms an specific English class schedule.

🧠 Logic
The schedule is determined using conditional statements (if, else if, else):

Kinder → Monday & Wednesday (4:00 PM – 5:00 PM)
1st Year → Tuesday & Thursday (4:30 PM – 5:30 PM)
2nd Year → Tuesday & Thursday (5:30 PM – 7:00 PM)
3rd Year → (custom schedule depending on your logic)
💻 Technologies Used
Java
Scanner (for user input)
If-Else statements
